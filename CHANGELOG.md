## [1.0.0] - 2026-02-16

### Added
- Initial open-source release of `ah-premium-lab`.
- Streamlit dashboard for A/H premium overview and pair detail analysis.
- Statistical diagnostics including ADF, Engle-Granger, half-life, rolling cointegration, and break detection.
- Backtesting toolkit with executability constraints and cost sensitivity reporting.
- GitHub Pages demo assets and report publishing pipeline.
